# Eteon-dl
An automated python script to download any YouTube playlist, no matter the size - In Seconds.


## ⚙️ Installation Guide (Windows)


Firstly, Copy and Paste main.py in your Venv environment
Then, Open **PowerShell** or **Terminal** in VS Code and run these commands in succession:

pip install yt-dlp

winget install ffmpeg

python main.py

Then paste your YouTube playlist URL when prompted (Playlist must be public/unlisted)

## 📌 Notes

- This guide is for **Windows OS** only
- Commands are executed in **VS Code Terminal** or **PowerShell**
- Downloaded files will be saved in the `./downloads/` folder within the same folder with thumbnails.
